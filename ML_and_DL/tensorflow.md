## 神经网络
- 层
    - 全连接(Fully connected)层: Dense -> 2D张量
    - 循环(Recurrent)层: LSTM -> 3D张量
    - 卷积(Concolution)层: Con2D -> 4D张量
- 损失函数
    - 二分类 -> 二元交叉熵 binary_crossentropy
    - 多分类 -> 分类交叉熵
        - 分类标签(one-hot) + `categorical_crossentropy`
        - 整数标签 + `sparse_categorical_crossentropy`
    - 回归 -> 均方误差
    - 序列学习 -> 联结主义时序分类(CTC)
- 定义模型
    - Sequential类
    - 函数式API
